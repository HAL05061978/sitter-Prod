## The Fix in Plain English

**When a counter is declined:**

1. Remove the rescheduler's child from ALL blocks (providing and needed)
2. Look at every providing block:
   - If it has zero children â†’ DELETE
   - If it only has the provider's own child â†’ DELETE
   - If it has other children â†’ KEEP
3. Look at every needed block:
   - If it has zero children â†’ DELETE
   - If it's missing the needing parent's own child â†’ DELETE
   - If it has the needing parent's child â†’ KEEP

**Result:** Only meaningful blocks remain on calendars!
